Automatic Installation

If you are on Mac or Linux you can use the following installation script to automatically download and install Theme Kit for you. You will need to make a small change to your shell initialization file (i.e. .bash_profile) before theme will show up in your console:

curl https://raw.githubusercontent.com/Shopify/themekit/master/scripts/install | python

An automated installer for Windows will be coming soon.

Manual Installation Guide

Download and Unzip the latest release from Github

If you are using Mac OS or Linux follow these instructions

If you are using Windows follow these instructions

If you want to install from source follow these instructions

Installing on Mac OS & Linux

  • Create a directory called Applications/bin. You can enter the command in a terminal window to create it:

    mkdir -p ~/Applications/bin

  • Open the ~Applications/bin directory and move the downloaded theme program into it. On Mac OS you can enter the following command to open both teh the bin and Download directories.

    open ~/Applications/bin && open ~/Downloads
  • Open your configuration file for your terminal interface:
    • If you are using Bash your file will be called .profile or .bash_profile
    • If you are using ZSH your file will be called .zshrc
    • If you are using another shell you probably know how to do this already
  • At the bottom of your file add the following:
    • export PATH=~/Applications/bin:$PATH
    • Or do this in a single command, using echo (substituting in your shell’s configuration file as above):
    • echo "export PATH=~/Applications/bin:$PATH" >> ~/.bash_profile

    • Reload your shell by closing your terminal window and opening a new one

    • Verify that Theme Kit has been installed by typing theme --help and you should see output similar to this.

    Installing on Windows

    • Create a folder inside C:\Program Files\ called Theme Kit
    • Copy the extracted program into C:\Program Files\Theme Kit
    • If the program is missing .exe at the end, rename the file and add the .exe file extension
    • Navigate to Control Panel > System and Security > System
      • Another way to get there is to Right-Click on My Computer and choose the properties item
    • Look for the button or link called Environment Variables
    • In the second panel look for the item called Path and double-click on it. This should open a window with a text field that is overflowing with content.
    • Move your cursor all the way to the end and add the following: ;C:\Program Files\Theme Kit\
    • Click OK until all the windows are gone.
    • To verify that Theme Kit has been installed, open cmd.exe and type in theme --help. You should see output similar to this.

    Install from Source

    Before you can get started you will need to install Go for your platform. With go installed, also verify go has been installed correctly.

    You will also need to have git installed in order to properly download themekit and it’s dependencies.

    You will need to add the bin directory within your go installation to your PATH. On a unix-like system using Bash, you’d do something like this:

    • export PATH=PATH-TO-YOUR-GO-FOLDER/bin:$PATH >> ~/.bash_profile
source ~/.bash_profile

    • Download themekit:

    go get github.com/Shopify/themekit

    Install themekit:

    go install github.com/Shopify/themekit/cmd/theme

    To verify that Theme Kit has been installed by typing theme --help. You should see output similar to this.
